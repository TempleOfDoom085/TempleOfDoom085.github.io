Knights Templar: The Siege of the Undead started as a Python assignment for IT 140 at Southern New Hampshire University. The brief was simple: build a text-based adventure with rooms, items, and win/lose conditions.

What it became is something much larger. Over seven major versions, the game grew from a terminal script into a full RPG with hand-coded SVG artwork, procedural Web Audio music, stat-based combat with equippable relics and special abilities, a quest system with NPC missions, a 3-phase cinematic boss fight, random world events with a wandering merchant, XP and leveling, and an achievement system culminating in a Saint Michael the Archangel divine armor cutscene.

The game is built entirely without external libraries — no game engine, no asset files, no audio downloads. Every sound is synthesized live using the Web Audio API. Every piece of artwork is hand-written SVG code following a strict cinematic dark-fantasy art direction. The hybrid image engine allows AI-generated photorealistic backgrounds to be layered in seamlessly.

The long-term goal is a full open-world 3D game. This browser version is the foundation — the design document, the lore, the combat mechanics, and the vision are all already built and battle-tested.
